# GlassFree3D_API
使用WebGPU实现裸眼三维内容的展示
## 研发计划
1.实现web版的裸眼3D播放器，直接加载网络视频，进行裸眼3D的播放

2.实现2D视频转裸眼3D视频的功能

3.实现视频生成裸眼3D图片的功能
